# Password Generator

A responsive and aesthetic password generator built using React and Tailwind CSS.

## Features

- Generate secure random passwords
- Adjustable password length
- Numbers toggle
- Special characters toggle
- Copy password to clipboard
- Save generated passwords
- Responsive pastel UI

## Tech Stack

- React
- Tailwind CSS
- JavaScript
